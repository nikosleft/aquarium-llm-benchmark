LLM Aquarium Benchmark

A comparative analysis of autonomous code generation capabilities across different Large Language Models running an HTML5 Canvas task.

1. Claude (3.5 Sonnet)

Exceptional visual fidelity, smooth delta-time turning, and high-performance offscreen background rendering.

~13 KB

2. Gemini (1.5 Flash)

Advanced biological movement (glide/thrust cycle), group separation physics, and three distinct fish species.

~16 KB

3. Claude-Genesis (Qwen35B)

Successfully resolved the corner-lock bug using coordinate targeting, but turn rates are frame-rate dependent.

~14 KB

4. ChatGPT (GPT-4o-Mini)

Highly lightweight structure. Implemented basic vector steering but features minimalist visual details.

~6 KB

5. HauhauCS (Normal)

Visual elements like swaying light caustics, but suffers from terminal corner trapping physics.

~8 KB

6. HauhauCS (Control)

High visual detail. Features a missing render parameter bug (invisible seaweed) and corner-lock physics.

~25 KB
